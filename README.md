# MatchWise
MatchWise is an AI-powered career coach that compares a user's resume with a job description.

## Planned Features

- Resume Upload
- Job Description Input
- Resume-to-job match score
- Matched and missing skills
- Resume Improvement Suggestions
- Interview Preparation
- Analysis History

## Planned Technology

- Frontend : React
- Backend : Python FastAPI
- Database : PostgreSQL
- AI: Azure OpenAI
- Deployment: Azure
